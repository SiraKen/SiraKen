GitHub OrganizationでVerifyする

GitHubのOrganizationアカウントといえば、こんなマークを見かける人も多いでしょう。

緑色のVerifiedバッジ。この“Verified”を付与してもらうには「ドメインの検証」を行う必要があります。詳しくは以下のページをご覧ください。

Organization のドメインを検証する

Organization が管理しているドメインを検証して、GitHub で Organization のアイデンティティを確認できます。

help.github.com

作業を大雑把に説明すると、ドメインの所有を確認するためにDNS設定のTXTレコードに指定された値をセットするというもの。 一度ドメインの検証を行ったのですが、なぜかVerifiedが付きませんでした。どうにか解決すべく色々な部分をチェックすると気になる点が。

GoogleのGitHub Orgページを見ると、opensource.googleとgoogle.comのドメイン2つが検証済みになっていました。

www.novalumo.llcのみ検証済みになっていましたが、www無しも認証してみるとVerifiedになりました。

設定されているWebサイトのURL、メールアドレスのドメインを確認する必要があるようです。両方のドメインを登録するとGitHubから確認のメールも届き、これで完了です。
